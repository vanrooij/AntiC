initialize

()

initialize determines the current platform that the application is running on and will immediately execute the onDeviceReady if we detect we're running on webOS, otherwise binds the onDeviceReady if we detect we're on a mobile device. This ensures that onDeviceReady gets executed regardless of what platform the app is being run on.

()
