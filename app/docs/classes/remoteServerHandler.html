remoteServerHandler Class

remoteServerHandler class which handles the direct interaction with the server. The communication is done via jQuery's .ajax calls

Methods

checkForUpdates

() Boolean

Uses an ajax call to the AreUpdates server API function to see if there are new updates since the last time the medical data was updated

Returns:

Boolean:

Whether or not there are valid updates available for download

fetchData

() String

fetchData uses an async ajax call to download all of the medical updates since the last time a download occurred

Returns:

String:

Stringify'd JSON of the medical information

isReachable

() Boolean

Uses an async ajax call to see if there is a valid connection to the remoteServer (i.e. if the device has internet enabled, and the remote server is accessible)

Returns:

Boolean:

Returns True if a connection can be established
